Projects - Personal Assistant Bot

I designed and developed a fully functional personal assistant bot powered by ChatGPT's API and Raspberry Pi. The bot features a custom-built body with four-wheel mobility for smooth movement. It uses a microphone to capture user commands and a display screen for visual responses, enabling an intuitive and interactive experience. The project involved integrating hardware and software to create an AI-driven system capable of real-time assistance. This hands-on experience allowed me to enhance my skills in robotics, programming, and AI, demonstrating my ability to create practical solutions that blend innovation with technology.
